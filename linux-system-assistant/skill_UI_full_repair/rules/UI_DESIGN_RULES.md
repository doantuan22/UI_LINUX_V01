# UI Design Rules

1. UI phải ưu tiên đọc rõ hơn là trong suốt đẹp mắt.
2. Glass panel không được quá trong khi dùng wallpaper phức tạp.
3. Text trắng/xám phải có đủ contrast.
4. Mỗi panel cần có title rõ.
5. Action row cần có icon, title, subtitle và trạng thái hover.
6. Không dùng quá nhiều màu neon cho text bình thường.
7. Accent color chỉ dùng cho trạng thái, icon, gauge hoặc active state.
8. Nếu label tiếng Việt quá dài, ưu tiên tăng width hoặc dùng tooltip, không để cắt vô nghĩa.
9. Tất cả padding/spacing/radius nên lấy từ `Metrics.qml`.
10. Tất cả màu nên lấy từ `Theme.qml`.
