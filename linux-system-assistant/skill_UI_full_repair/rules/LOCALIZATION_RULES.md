# Localization Rules

Ưu tiên tiếng Việt cho UI.

## Mapping bắt buộc

- `Performance Mode` -> `Chế độ hiệu năng`
- `Download` -> `Tải xuống`
- `Upload` -> `Tải lên`
- `Monitor • Optimize • Protect` -> `Giám sát • Tối ưu • Bảo vệ`
- `System Tools` -> `Công cụ hệ thống`
- `Process Manager` -> `Quản lý tiến trình`
- `Settings` -> `Cài đặt`
- `Quick Actions` -> `Thao tác nhanh`
- `Error Log` -> `Log lỗi`
- `Exit` -> `Thoát`
- `Minimize` -> `Thu gọn`
- `Clean Cache` -> `Dọn dẹp hệ thống`
- `Hardware Info` -> `Thông tin phần cứng`

## Tên riêng

Có thể giữ nguyên:
- System Assistant
- CPU, GPU, RAM
- NVIDIA, AMD, Intel
- tên ứng dụng/process
