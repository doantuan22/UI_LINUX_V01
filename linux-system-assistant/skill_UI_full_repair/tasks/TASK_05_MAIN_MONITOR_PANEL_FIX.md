# TASK 05 - Sửa Main Monitor Panel

## Mục tiêu

Main panel cân đối, không thừa khoảng trống lớn, không còn shortcut buttons dưới cùng.

## Việc cần làm

1. Bỏ các nút:
   - Kiểm tra hệ thống
   - Dọn dẹp nhanh
   - Quản lý tiến trình
   - Cài đặt
2. Căn lại chiều cao panel sau khi bỏ nút.
3. Giữ các phần:
   - Header
   - Tổng quan hệ thống
   - 6 stat cards
   - Network card
   - Chế độ hiệu năng
4. Sửa stat cards:
   - GPU name không hiển thị quá dài.
   - Có short name, ví dụ `RTX 4060`.
   - Value text lớn hơn subtext.
5. Sửa network:
   - `Download` -> `Tải xuống`
   - `Upload` -> `Tải lên`
6. Sửa performance:
   - `Performance Mode` -> `Chế độ hiệu năng`
   - Inactive button không quá mờ.

## Acceptance criteria

- Main panel không còn khoảng trống thừa lớn.
- Không còn shortcut buttons dưới cùng.
- Các stat card đọc rõ.
- GPU text không bị cắt khó hiểu.
