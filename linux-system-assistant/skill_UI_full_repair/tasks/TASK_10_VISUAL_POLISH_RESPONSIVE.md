# TASK 10 - Visual polish và responsive

## Mục tiêu

Sau khi sửa từng panel, căn lại toàn bộ UI cho cân đối.

## Việc cần làm

1. Kiểm tra spacing giữa các panel.
2. Các panel cùng hàng nên có top/bottom alignment hợp lý.
3. Radius, border, shadow nhất quán.
4. Hover state cho action row/button.
5. Active state rõ nhưng không lòe loẹt.
6. Căn responsive theo screen:
   - Không vượt màn hình.
   - Nếu màn nhỏ, giảm scale hoặc cho overlay scroll.
7. Không để panel bị dính sát mép màn hình.

## Acceptance criteria

- UI nhìn đồng nhất.
- Không còn chỗ quá chật hoặc quá trống.
- Dùng được trên màn hình laptop phổ biến.
