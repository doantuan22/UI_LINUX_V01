# TASK 00 - Backup và audit UI hiện tại

## Mục tiêu

Trước khi sửa, AI coding phải hiểu cấu trúc QML hiện tại và không phá backend.

## Việc cần làm

1. Liệt kê các file QML đang tạo UI.
2. Tìm file chứa:
   - Floating icon.
   - Dashboard/overlay.
   - Main monitor panel.
   - System tools panel.
   - Process panel.
   - Bottom panels.
   - Theme/Metrics.
3. Tạo ghi chú ngắn vào `reports/ui_audit.md`:
   - file nào phụ trách phần nào.
   - panel nào đang overflow.
   - component nào có thể tái sử dụng.
4. Không sửa code ở task này trừ khi tạo report.

## Acceptance criteria

- Có danh sách file UI chính.
- Có ghi chú panel/component hiện tại.
- Biết chính xác nơi cần sửa cho các task sau.
